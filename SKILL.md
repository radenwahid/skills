# OpenClaw Workflow Visualizer Dashboard

## Description
This is a Premium GUI Dashboard skill for OpenClaw that visualizes Agent workflows in real-time. It reads Markdown steps and converts them into a React Flow diagram.

## Features
- Real-time bot progress tracking via API Polling.
- Dark mode, glassmorphism UI.
- Displays all active skills and running workflows.

## Installation / Usage
This skill is a React web application.
1. Run `npm install`
2. Configure `src/config.js` with your API URL.
3. Run `npm run build` to generate the production GUI.
4. Serve the `/dist` folder.
